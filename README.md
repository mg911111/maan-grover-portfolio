# Maan Grover — Interactive 3D Portfolio

An interactive 3D portfolio built with React, Vite, Three.js, and modern web animation.

The experience includes a cinematic portfolio hub, a drivable 3D world, an animated avatar, project showcases, resume highlights, contact links, day/night mode, voiceover, ambient music, and custom UI sound effects.

## Experience

Visitors can choose between two paths:

- Drive the 3D World — explore portfolio stops inside an interactive city environment.
- Browse Portfolio Hub — view a cinematic portfolio website with projects, skills, experience, resume, and contact information.

## Features

- Interactive 3D driving portfolio world
- Cinematic portfolio hub
- Animated 3D avatar hero section
- Project, skills, experience, resume, and contact sections
- Day/night mode
- Fullscreen support
- Voiceover intro and ambient audio
- Click, hover, and panel sound effects
- Responsive visual design

## Tech Stack

- React
- Vite
- Three.js / React Three Fiber
- JavaScript
- CSS
- GLB / 3D assets
- Custom audio assets

## Local Development

Start the dev server:

npm run dev

Open:
